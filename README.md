# AIOT-Balloon-Game


> AIOT-17 『英文單字射氣球』
> 
> 小弟與另一位組員一同製作的英文單字射氣球遊戲。

----------

    開始時螢幕左上角會出現圖片(例如BOY)，
    畫面中央會由下往上飄入三顆氣球，
    分別代表B、O、Y，按照順序點擊會顯示答對，
	答錯會於下方顯示正確答案。

----------

    使用技術：
    小遊戲：html、JavaScript、jQuery

----------
